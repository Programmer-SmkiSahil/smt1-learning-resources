Box Model

Box Model ada 4 Macam

  1. Content : isi utama
  2. Padding : Jarak content ke border
  3. Border : Garis luar Content
  4. Margin : Jarak antar box

Padding

Padding ada 4 Macam : Top, Right, Bottom, Left

Top untuk Atas

Right untuk Kanan

Bottom untuk Bawah

Left untuk Kiri

GABUNGAN

1. Satu Value untuk semua sisi

GABUNGAN 2 VALUE

2. Value Pertama untuk Top dan Bottom. Value kedua untuk Right dan Left

TIGA VALUE

3. Value pertama Top. Value kedua Right & Left. Value ketiga Bottom

EMPAT VALUE

4. Value pertama Top. Value kedua Right. Value ketiga Bottom. Value keempat Left

Margin

Margin ada 5 Macam : Top, Right, Bottom, Left, Auto

Top untuk Atas

Right untuk Kanan

Bottom untuk Bawah

Left untuk Kiri

Auto untuk Rata Kanan Kiri

GABUNGAN

1. Satu Value untuk semua sisi

GABUNGAN 2 VALUE

2. Value Pertama untuk Top dan Bottom. Value kedua untuk Right dan Left

TIGA VALUE

3. Value pertama Top. Value kedua Right & Left. Value ketiga Bottom

EMPAT VALUE

4. Value pertama Top. Value kedua Right. Value ketiga Bottom. Value keempat Left
