Flexbox

Flexbox digunakan untuk menyusun letak elemen agar tetap rapi dan responsive.

Flexbox punya 2 bagian Utama : Container dan Item.

Properti Container:

  1. flex-direction: row, column
  2. justify-content: flex-start, center, space-between, space-around
  3. align-items: stretch, flex-start, flex-end, center
  4. gap: 5px, 10px, 15px
  5. flex-wrap: wrap, no-wrap

Properti Item:

  1. flex: 1 1 100px
  2. align-self: flex-start, flex-end, center
  3. order: 1
