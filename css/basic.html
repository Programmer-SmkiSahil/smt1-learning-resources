Ada 3 Cara penulisan CSS

  1. Inline (atribut Tag HTML)
  2. Internal (di dalam Tag Head)
  3. External (di luar HTML, tapi dipanggil di Tag Head)

Struktur Dasar CSS

  1. Selector : Kata kunci untuk elemen yang akan diberi style
  2. Property : Nama style yang akan diberikan
  3. Value : Nilai yang diinginkan untuk style
